機械学習の Python との出会い (Machine Learning Meets Python)
============================================================

* **Author** : [神嶌 敏弘 (Toshihiro Kamishima)](http://www.kamishima.net)

機械学習と Python プログラミングについて初歩的な知識を前提として，NumPy をはじめとする科学技術計算関連のモジュールの使い方を説明するチュートリアルです．
