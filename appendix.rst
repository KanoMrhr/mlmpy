.. _appendix-import:

モジュールのimport
------------------

.. index::
   single: import

本稿のコードは，モジュールが以下のように ``import`` されていることを前提としています．::

  import numpy as np
  import scipy as sp
  import matplotlib.pyplot as plt
  import sklearn
